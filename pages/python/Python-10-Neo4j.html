# 10. Neo4j 图数据库

# 1. 安装配置

  1. 从neo4j官网下载安装包
  2. 配置JDK环境，比如 neo4j v4.0.3 需要jdk11
  3. 离线安装包解压即可
  4. 启动数据库

    1. 在../neo4j-community-4.0.3/bin安装目录下打开命令行

      neo4j console
      1

    2. 正常数据库打开效果
